## 🔑 Key Topics Covered

### 🎣 Types of Phishing
- **Email Phishing** — Mass fraudulent emails mimicking trusted sources
- **Spear Phishing** — Targeted attacks on specific individuals
- **Whaling** — High-level attacks targeting executives/CEOs
- **Smishing** — Phishing via SMS text messages
- **Vishing** — Phishing through voice/phone calls
- **Clone Phishing** — Duplicating legitimate emails with malicious links
- **Pharming** — Redirecting users to fake websites via DNS manipulation

### 🚩 Red Flags to Watch For
- Urgent or threatening language
- Mismatched or suspicious URLs
- Generic greetings ("Dear User")
- Unexpected attachments
- Requests for sensitive information
- Poor grammar and spelling errors

### 🛡️ Prevention & Best Practices
- Verify sender identity before clicking links
- Hover over URLs to check the actual destination
- Enable Multi-Factor Authentication (MFA)
- Keep software and antivirus updated
- Report suspicious emails to IT/Security teams
- Use email filtering and anti-phishing tools

### 🏢 Organizational Defenses
- Employee security awareness training
- Simulated phishing campaigns
- Email gateway filtering (SPF, DKIM, DMARC)
- Incident response planning
- Zero Trust Security Model
